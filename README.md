# 星途 · StarPath v3 — 爱豆应援模拟器

移动端优先 · 不限制内存 · 无限制上传 · 离线版

## 功能

- 爱豆养成模拟（热度/人气/好感/等级）
- 写真相册（导入/排序/放大查看/删除）
- 视频播放
- 粉丝留言互动
- 成就系统
- 每日任务
